# 本地文件到知识资产工作流

## 场景

很多人的电脑里有大量真实资料：

- 项目文档
- 会议纪要
- CSV 数据
- 历史方案
- 竞品拆解
- 聊天记录
- 代码片段

这些东西如果只是躺在文件夹里，就不是资产。

这个 workflow 的目标是：把本地文件整理成可复用的知识资产，用于写文章、做知识库、生成视频和沉淀方法论。

## Workflow

```text
文件夹扫描
  -> 文件分类
  -> 摘要提取
  -> 场景归档
  -> 可复用观点提炼
  -> 风险脱敏
  -> 生成知识卡片
  -> 入库
```

## AI 适合做什么

- 摘要
- 分类
- 提炼观点
- 生成知识卡片
- 发现内容选题

## 代码和规则必须做什么

- 文件遍历
- 去重
- 过滤大文件
- 敏感词检查
- 路径记录
- 输出结构化索引

## 输出

- 文件索引
- 主题分类
- 可复用素材卡
- 文章选题
- 知识库章节建议
- 需要脱敏的内容清单

## 适合接入

- Codex：本地文件扫描和整理
- Claude SDK：深度文件处理和工具调用
- 飞书：知识库入库
- GitHub：开源模板和项目文档沉淀
